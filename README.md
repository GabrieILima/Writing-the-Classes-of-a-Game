# 🎮 Writing the Classes of a Game

Modelagem de classes para um jogo usando Programação Orientada a Objetos (POO) em JavaScript, desenvolvido como terceiro desafio do curso de Lógica de Programação da [DIO.me](https://dio.me).

---

## 📌 Sobre o projeto

O projeto cria uma classe `Hero` com os atributos `nome`, `idade` e `tipo`, e um método `attack()` que utiliza `switch/case` para exibir o ataque correspondente ao tipo do herói.

| Tipo | Ataque |
|---|---|
| Mago | magia |
| Guerreiro | espada |
| Monge | artes marciais |
| Ninja (default) | shuriken |

**Exemplo de saída:**
```
O Mago Gabriel atacou usando magia
O Guerreiro Jojiberto atacou usando espada
O Monge Jubriscreide atacou usando artes marciais
O Ninja Naruto atacou usando shuriken
```

## ▶️ Como executar

1. Clone o repositório:
```bash
git clone https://github.com/GabrieILima/Writing-the-Classes-of-a-Game.git
```
2. Abra o arquivo no editor de sua preferência
3. Execute com Node.js:
```bash
node classes-game.js
```

---

## 💡 O que aprendi

- Criação de classes com `class` e uso de `constructor` em JavaScript
- Definição de métodos dentro de uma classe (`attack()`)
- Uso de `switch/case` para controle de fluxo por tipo
- Instanciação de múltiplos objetos a partir de uma mesma classe
- Conceitos fundamentais de Programação Orientada a Objetos (POO)
